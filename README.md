**독서 기록을 간편하게!**

책을 읽으며 감동적인 구절을 만났을 때, 마음 깊이 새기고 싶지 않으신가요?

**독기**는 당신이 읽은 책 속 구절을 간편하게 기록하고, 특별한 순간을 영원히 간직할 수 있도록 도와줍니다. 마음에 드는 구절을 기록하고, 얼마나 기록했는지 레벨을 통해 알아보세요!
